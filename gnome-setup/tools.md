# via `dnf`

- **git-delta** — A nicer `git diff`/`git show` pager with syntax highlighting and better formatting.
- **ddclient** — Dynamic DNS updater; keeps DNS records updated when your public IP changes (often used with home connections).
- **cloudflared** — Cloudflare Tunnel client; exposes local services securely through Cloudflare and can run DNS-over-HTTPS.
- **pavucontrol** — PulseAudio/PipeWire volume control GUI; manage inputs/outputs, per-app volumes, and audio routing.
- **fastfetch** — System info fetch tool (like neofetch); prints OS/hardware details in the terminal.
- **ani-cli** — CLI tool to search and stream anime from the terminal (uses online sources).
- **memtest86+** — Bootable RAM testing tool to check for memory errors.
- **fdfind** — Faster, simpler alternative to `find` with good defaults and ignore rules.
- **bat** — `cat` replacement with syntax highlighting, paging, and Git integration.
- **duf** — Disk usage/free utility (like `df`) with a nicer, more readable output.
- **btop** — Interactive resource monitor (CPU/RAM/disk/net/processes), modern alternative to `top`.
- **yazi** — Fast terminal file manager with previews and keyboard-driven navigation.
- **f3** — “Fight Flash Fraud”; tests USB/SD storage for fake capacity and checks read/write reliability.
- **ddcutil** — Control external monitors via DDC/CI (brightness, contrast, input source) from Linux.
- **stress-ng** — System stress testing and benchmarking tool (CPU, memory, I/O, etc.).
- **wget** — Command-line downloader for HTTP/HTTPS/FTP.
- **ifuse** — Mount iPhone/iPad storage via FUSE using libimobiledevice (access files on iOS devices).

# via `npm` (global)

- **vercel** — Vercel CLI for deploying and managing projects (deployments, environment variables, logs, and project settings).
- **aicommits** — CLI for generating concise Git commit messages automatically from your staged changes using AI.

# via `brew`

**tlrc** — Community `tldr` client; shows short, practical command examples (a simplified “cheat sheet” for common CLI commands).

- **viu** — Displays images directly in the terminal (using terminal graphics protocols where supported).
