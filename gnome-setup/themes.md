This document tracks the themes used in my GNOME environment.

| Type       | Theme Name    |
| ---------- | ------------- |
| GTK (Apps) | WhiteSur-Dark |
| Shell      | WhiteSur-Dark |
| Icons      | Papirus-Dark  |
| Cursor     | Bibata-Jaroo  |

---

## Verify

```bash
gsettings get org.gnome.desktop.interface gtk-theme
gsettings get org.gnome.desktop.interface icon-theme
gsettings get org.gnome.desktop.interface cursor-theme
```
