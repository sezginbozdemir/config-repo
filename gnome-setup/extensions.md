This is a list of GNOME extensions I use, for easy setup on a new machine.

| Name                 | Description / Notes                     | GNOME Extensions Link                                                           |
| -------------------- | --------------------------------------- | ------------------------------------------------------------------------------- |
| AppIndicator Support | Enables system tray indicators for apps | [Link](https://extensions.gnome.org/extension/615/appindicator-support/)        |
| App Menu is Back     | Restores application menu in top bar    | [Link](https://extensions.gnome.org/extension/6433/app-menu-is-back/)           |
| App Menu             | Application menu in top bar             | [Link](https://extensions.gnome.org/extension/6/applications-menu/)             |
| Background Logo      | Custom logo in the background           | [Link](https://extensions.gnome.org/extension/889/background-logo/)             |
| Dash to Dock         | Replaces the dash with a dock           | [Link](https://extensions.gnome.org/extension/307/dash-to-dock/)                |
| Just Perfection      | All-in-one UI tweaks                    | [Link](https://extensions.gnome.org/extension/3843/just-perfection/)            |
| Logo Menu            | Custom logo menu in top bar             | [Link](https://extensions.gnome.org/extension/4451/logo-menu/)                  |
| Paper WM             | Window management tweaks                | [Link](https://extensions.gnome.org/extension/6099/paperwm/)                    |
| Quick Settings       | Quick settings for audio input/output   | [Link](https://extensions.gnome.org/extension/5940/quick-settings-audio-panel/) |
| Simple Timer         | Countdown / timer on top bar            | [Link](https://extensions.gnome.org/extension/5115/simple-timer/)               |
| User Themes          | Allows loading custom GTK themes        | [Link](https://extensions.gnome.org/extension/19/user-themes/)                  |
| Vitals               | System monitoring (CPU/RAM/etc.)        | [Link](https://extensions.gnome.org/extension/1460/vitals/)                     |
