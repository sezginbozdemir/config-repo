This document lists the main applications installed and used on my Fedora system.

## Terminal

- **Kitty**  
  Fast, GPU-accelerated terminal emulator.

## Diagram & Design

- **Draw.io (diagrams.net)**  
  Used for creating flowcharts, system architecture diagrams, and visual documentation.

## Notes & Knowledge Base

- **Obsidian**  
  Markdown-based note-taking and knowledge management system.

## Web Browser

- **Brave**  
  Privacy-focused Chromium-based web browser.

## Cloud Gaming

- **GeForce NOW (GeForce Infinity)**  
  Cloud gaming platform for streaming games.

## Productivity & Search

- **Albert**  
  Application launcher and productivity tool.

## Communication

- **Discord**  
  Messaging, voice, and community communication platform.

## Downloads

- **qBittorrent**  
  Open-source BitTorrent client.

## Media Player

- **SMPlayer (with mpv backend)**  
  Media player with advanced playback features using mpv engine.
